Premium Saudi real-estate investment platform illustration, architectural pencil sketch, fine

invent or recreate partner logos.

09 - QUALITY REQUIREMENTS
Responsive, Accessibility, SEO, and Performanceمتطلبات الجودة
Responsive rules
Desktop: 4 cards in assets and
contributions when space allows.
Tablet: 2 cards per row or horizontal
carousel.
Mobile: 1 card per row, full-width
search fields, off-canvas menu.
Hero illustrations should crop
gracefully without hiding headline.
Dark contributions panel must keep
internal padding on small screens.
Accessibility
Use semantic HTML: header, nav,
main, section, article, footer.
All buttons and accordions keyboard
accessible.
Contrast must pass WCAG AA,
especially gold on ivory and navy.
Images need meaningful Arabic alt
text.
Forms require labels, aria-describedby,
and clear errors.
SEO
Arabic metadata for every route.
OpenGraph image for landing and
article pages.
Structured data for articles and
organization if appropriate.
Clean slugs and canonical URLs.
HTML text for all headings instead of
baked-in image text.
Performance
Use next/image for all illustrations.
Serve responsive image sizes.
Lazy-load below-the-fold images.
Keep animation subtle and GPU-
friendly.
Target Lighthouse 90+ across major
categories.

10 - ACCEPTANCE CRITERIA
Final Delivery Checklistمعايير القبول
The final build is accepted only if:
It clearly resembles the supplied Mahabah visual direction.
The UI is Arabic-first and RTL-first without awkward alignment.
The hero feels cinematic and premium, not stock-photo based.
The assets and contributions sections use reusable card components.
The dark contributions band has a high-trust investment dashboard feel.
All image assets share one sketch style and contain no baked-in text.
Forms have validation, loading, and success states.
The footer includes legal, services, support, social, and contact information.
Mobile design is polished and intentional.
Code is typed, componentized, maintainable, and CMS-ready.

You are a senior product designer, UX architect, Arabic RTL front-end engineer, and AI image 
director.
Build a production-ready Saudi Real Estate Contribution Platform named " "مهابةfor managing, 
studying, structuring, and presenting real-estate assets and real-estate contributions.
Use the attached design reference report and cropped segment images as the visual source of 
truth. The goal is not a generic website. The result must feel like a premium institutional Saudi 
real-estate investment platform with governance, credibility, and high trust.
PROJECT POSITIONING
- Brand name: مهابة
- English transliteration: Mahabah
- Descriptor: إلدارة المساهمات العقارية
- Market: Saudi Arabia
- Direction: Arabic-first, RTL-first
- Tone: institutional, regulated, premium, calm, trustworthy
- Visual DNA: architectural pencil sketches, ivory surfaces, dark navy investment panels, muted 
gold accents, thin borders, elegant shadows, high whitespace.
TECH STACK
- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React icons, customized where needed
- Fully responsive, mobile-first
- Arabic RTL layout by default
- SEO metadata and OpenGraph images
- Component-driven, data-driven, CMS-ready
COLOR SYSTEM
- Ivory background: #F7F3EE
- Warm surface: #EFE8DE
- Primary navy: #071A2D
- Secondary navy: #0D243D
- Gold: #A87C4F
- Light gold: #C7A27A
- Body text: #1F1F1F
- Muted text: #6B6259
TYPOGRAPHY
- Use a premium Arabic font such as IBM Plex Sans Arabic, Noto Kufi Arabic, Noto Sans Arabic, or 
DIN Next Arabic if available.
- Headings should feel formal and investment-grade.
- Body text should be readable and not too thin.
- Use Arabic numerals or localized formatting consistently across metrics.
GLOBAL LAYOUT REQUIREMENTS
- Header is sticky.
- Logo is on the right.
- Navigation is RTL.
- Main CTAs are on the left on desktop.
- Use max-width content containers but allow selected sections, especially hero and dark 
contribution band, to feel wide and cinematic.
- Use thin #E1D6C9 borders, 16px to 28px border radii, soft shadows, and muted gold dividers.
- Avoid colorful SaaS styling, gradients, playful illustrations, generic stock photos, or 
startup-style cards.
